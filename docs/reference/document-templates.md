## 1. 文档目标

本文档用于给 `POMS` 当前文档体系提供统一模板，降低新建文档时的结构漂移和页头格式漂移。

本文档当前覆盖三类常见文档：

- `docs/design/` 下的设计文档
- `docs/modules/` 下的模块设计文档
- `docs/reference/` 下的参考资料文档

---

## 2. 通用页头模板

除目录入口型 `README.md` 外，正式文档建议统一采用以下页头：

```md
# 文档标题

**文档状态**: Draft
**最后更新**: 2026-03-16
**适用范围**: `POMS` 第一阶段某个域或某类参考资料
**关联文档**:

- 上游设计:
  - `上游文档路径`
- 同级设计:
  - `同域或同类文档路径`
- 相关 ADR:
  - `相关 ADR 路径`

---
```

说明：

- `最后更新` 使用 `YYYY-MM-DD` 格式
- 若某一分组当前为空，可以省略该分组
- 页头列表统一使用空格缩进，不混用制表符

## 6. 目录入口 README 模板

适用于各层目录入口，如 `docs/design/README.md`、`docs/modules/README.md`、`docs/reference/README.md`。

推荐骨架如下：

```md
# 目录标题

本目录用于说明：

- 目录职责
- 适合放什么内容
- 不适合放什么内容
- 当前文档清单
- 命名规则与维护约定

## 目录定位

## 当前文档

## 命名建议

## 使用建议

## 维护约定
```

说明：

- 目录入口 README 一般不强制要求页头元信息
- 目录入口更偏索引和治理说明，不承担完整业务结论

---

## 7. 使用约定

- 新建文档时，优先从本模板复制，再按目录类型裁剪
- 如果文档已经存在成熟结构，不应为了套模板而破坏有效内容组织
- 模板用于收敛格式，不用于限制必要的专业内容展开
