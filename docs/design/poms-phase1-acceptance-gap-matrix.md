## 2. 审查口径

本轮审查采用以下固定口径：

1. 以路线图中的第一阶段承诺为主口径。
2. 设计文档只用于解释承诺边界，不单独扩大范围。
3. `Done` 需要同时具备实现证据和验收证据。
4. 若只具备实现证据、不具备域级验证证据，应降为 `In Review` 或 `Reopened`。
